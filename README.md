IGP4IL Unity Helicopter Game Prototype

A simple 2D helicopter game prototype controlled through an Arduino setup.
The objective is to deliver a specified number of crates to a delivery zone while avoiding obstacles, managing fuel consumption, and completing deliveries as efficiently as possible to achieve a high rating at the end of the level.

Movement is controlled using physical inputs, with a joystick for horizontal movement and a potentiometer for vertical control.
Arduino outputs provide gameplay feedback, including the remaining number of crates displayed on a 7-segment display and the current fuel level represented by a servo-driven fuel gauge.
